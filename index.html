Negative Margin -- A way to center the content

Example Code:

<div class="content">
    <img src="http://upload.wikimedia.org/wikipedia/commons/1/1b/Square_200x200.png" width="200" height="100">
</div>

CSS

.Content {
    position:absolute;
    left:50%;
    top:50%;
    width:550px;
    height:350px;
    margin-top:-190px;
    margin-left:-275px;
    padding:15px;
}

Step One: Center the div but div center is not the same as body center
